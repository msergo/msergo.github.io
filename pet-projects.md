Some of my projects which were originally developed for fun, but later became useful so I keep maintaining them.

1. [A Telegram bot](https://github.com/msergo/eki_telegram_bot) for fetching translations from official Estonian 🇪🇪 [In active use](https://t.me/eki_ee_bot), for more than 5 years already 😎.
2. A service for generating disposable email aliases on the existing WildDuck email server instance. It was developed on the [K-Space hackathon 2023](https://wiki.k-space.ee/en/events/2023-hackathon). Code is available [here](https://git.k-space.ee/k-space/wildflock).
3. A simple web radio made with Raspberry Pi Zero and a Bluetooth speaker. I use it at home often as it's much more convenient compared to streaming from a laptop or a phone. [Code is here](https://github.com/msergo/mplayer-pi).
